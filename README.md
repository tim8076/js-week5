## 安裝

Node.js 版本建議為：v20.10.0 以上

### 取得專案

```
git clone https://github.com/tim8076/js-week5.git
```

### 移動到專案內

```
cd js-week5
```

### 安裝套件

```
npm install
```

### 運行專案

```
npm run dev
```

## 資料夾說明

- components - react 元件放置處
- assets - 靜態檔案放置處
  - scss - scss 檔案放置處
- data - data資料放置處

## 專案技術

- Node.js v20.10.0
- react 18.3.1
- bootstrap 5.3.3
